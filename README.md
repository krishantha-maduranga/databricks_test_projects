### Transformation Check List
- Removing Duplicates on a Column (key columns like ID, Email, Phone)
- Removing Leading and Trailing Spaces (Trim) on a column (string columns)
- Correcting Typos on a column (string columns)
- Normalising values to have Capital Letter for the first word in each word (initcap)
- Converting to the correct data type (string to Int, Float or Date)
- Handling missing values (Need the guidance from the domain expert or Business Team for this)
